Pool-Pro.uk pool scoring app for teams and singles.
Features:
player names
team names
data export
custom colours per player 
persistent data storage
data import
shot clock
match clock
device time
clear data
restore defaults
API - send data to server
results table and score keeping
epic auto sizing of long names to fit space
